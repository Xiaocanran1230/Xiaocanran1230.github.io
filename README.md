# Canran Xiao Academic Homepage

This is a static GitHub Pages personal homepage.

## Deploy

1. Create a GitHub repository named `your-github-username.github.io`.
2. Copy `index.html` and the `assets/` folder into the repository root.
3. Commit and push to the `main` branch.
4. Open `https://your-github-username.github.io/` after GitHub Pages finishes building.

## Files

- `index.html`: the homepage.
- `assets/profile-campus.png`: profile photo used by the homepage.
